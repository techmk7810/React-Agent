
# 🤖 ReAct AI Agent

An intelligent AI agent built using **Python**, **LangChain**, and **OpenAI** that follows the ReAct (Reasoning + Acting) framework to solve user queries by reasoning step-by-step and utilizing external tools.

## 🚀 Features

- 🧠 ReAct reasoning framework
- 💬 Natural language interaction
- 🔧 Tool calling using LangChain
- 🤖 OpenAI LLM integration
- 📚 Prompt engineering
- ⚡ Easy to run and customize

## 🛠️ Tech Stack

- Python
- LangChain
- OpenAI
- LangSmith
- Jupyter Notebook (Google Colab)

## 📂 Project Structure

```
ReAct-Agent/
│── ReAct_Agebnt.ipynb
│── README.md
│── requirements.txt
```

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/your-username/ReAct-Agent.git
cd ReAct-Agent
```

Install dependencies:

```bash
pip install -r requirements.txt
```

## ▶️ Run the Project

Open the notebook in **Google Colab** or **Jupyter Notebook** and execute all cells.

## 🔑 Environment Variables

Create a `.env` file or configure environment variables with your API keys.

Example:

```
OPENAI_API_KEY=your_api_key
LANGSMITH_API_KEY=your_langsmith_key
```
